# 图片尺寸编辑应用

## 项目介绍

这是一个图片尺寸编辑和排版工具，专为需要精确控制图片打印尺寸的用户设计。通过直观的界面，用户可以轻松设置画布大小、导入图片、标注实际尺寸，并进行排版和导出操作。
在线体验链接：https://tomatoouo.github.io/size-setting/

## 核心功能

### 1. 画布设置
- 支持多种预设尺寸（A6, A5, A4, A3, A2, A1, A0, Letter, Legal, Tabloid）
- 支持自定义画布尺寸（宽度和高度）
- 可调整PPI（像素每英寸），从72到600
- 可选择颜色模式（RGB, 灰度）
- 可设置背景颜色

### 2. 图片导入
- 支持拖拽图片到指定区域
- 支持通过文件选择器导入多张图片

### 3. 尺寸设置
- **使用标注线**：在图片上画一条线段，输入实际长度，自动计算图片尺寸
- **直接输入尺寸**：直接输入图片的宽度和高度（毫米），支持保持宽高比

### 4. 图片操作
- 支持旋转图片（每次90度）
- 支持复制图片
- 支持重新标注尺寸
- 支持删除图片
- 支持批量选择和操作

### 5. 排版功能
- **一键排版**：自动排列画布上的图片（优先放置大图片）
- 支持图片拖拽移动和位置调整
- 提供对齐辅助线

### 6. 撤销和回退
- **撤销**：撤销上一步操作
- **回退**：恢复已撤销的操作
- 操作状态实时反馈

### 7. 导出功能
- 导出为PNG或JPG图片
- 可调整JPG质量
- 导出为JSON文件
- 临时保存到本地存储

### 8. 视图控制
- 支持画布缩放（鼠标滚轮或缩放按钮）
- 支持画布拖拽移动
- 支持适应屏幕尺寸

## 使用方法

### 基本操作流程

1. **设置画布**
   - 在左侧面板选择预设尺寸或设置自定义尺寸
   - 调整PPI、颜色模式和背景颜色

2. **导入图片**
   - 拖拽图片到"拖拽图片到这里"区域
   - 或点击"选择文件"按钮选择图片

3. **设置图片尺寸**
   - **使用标注线**：
     - 在弹出的窗口中选择"使用标注线"
     - 在图片上点击并拖动画一条线段
     - 输入线段的实际长度（毫米）
     - 点击"确认添加"
   - **直接输入尺寸**：
     - 在弹出的窗口中选择"直接输入尺寸"
     - 输入图片的宽度和高度（毫米）
     - 可选择是否保持宽高比
     - 点击"确认添加"

4. **编辑图片**
   - 点击图片选中它
   - 使用图片工具栏进行旋转、复制、重新标注或删除操作
   - 拖拽图片可移动位置
   - 按住Shift键可选择多张图片

5. **撤销和回退**
   - 点击顶部的"撤销"按钮（向左箭头图标）撤销上一步操作
   - 点击顶部的"回退"按钮（向右箭头图标）恢复已撤销的操作

6. **一键排版**
   - 点击顶部的"一键排版"按钮，自动排列所有图片

7. **导出**
   - 点击顶部的"导出图片"按钮，选择格式和质量
   - 点击"导出JSON"按钮，导出为JSON文件
   - 点击"临时保存"按钮，保存到本地存储
   - 点击"读取临时保存"按钮，加载之前保存的内容

### 快捷键

- **Delete/Backspace**：删除选中的图片
- **Shift+点击**：多选图片
- **鼠标右键**：拖拽移动画布
- **鼠标滚轮**：缩放画布

## 本地运行

1. 克隆或下载项目到本地
2. 打开项目目录
3. 启动本地服务器：
   - 使用Python：`python -m http.server 8080`
   - 或使用其他本地服务器工具
4. 在浏览器中访问 `http://localhost:8080`

## 注意事项

- 建议使用现代浏览器（Chrome, Firefox, Edge）访问
- 图片尺寸设置时，建议使用标注线方式获得更准确的尺寸
- 导出图片时，较高的PPI会产生更大的文件大小
- 临时保存的数据存储在浏览器本地存储中，清除浏览器数据会丢失这些数据
- 撤销和回退功能仅记录图片操作，不包括画布设置的更改

## 常见问题

### Q: 图片导入后没有弹出尺寸设置窗口？
A: 请检查浏览器是否阻止了弹出窗口，或尝试刷新页面后重新导入。

### Q: 标注线设置尺寸时，线段长度不准确？
A: 请确保线段绘制时尽量保持直线，避免弯曲或倾斜。

### Q: 一键排版后图片位置不理想？
A: 一键排版是基于简单的网格排列算法，可能需要手动调整个别图片的位置。

### Q: 导出的图片质量不高？
A: 请在导出时选择较高的JPG质量，或选择PNG格式。

### Q: 撤销和回退按钮有时会隐藏？
A: 当没有可撤销的操作时，撤销按钮会隐藏；当没有可回退的操作时，回退按钮会隐藏。
