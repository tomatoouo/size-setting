# 图片尺寸编辑应用

## 项目介绍

这是一个用于编辑和设置图片打印尺寸的网页应用，支持通过标注线或直接输入尺寸的方式来设置图片大小，并提供一键排版、导出等功能。

## 功能说明

### 核心功能

1. **画布设置**
   - 支持多种预设尺寸（A4, A3, A2, A1, A0, Letter, Legal, Tabloid）
   - 支持自定义画布尺寸
   - 可调整PPI（像素每英寸）
   - 可选择颜色模式（RGB, 灰度）
   - 可设置背景颜色

2. **图片导入**
   - 支持拖拽图片到指定区域
   - 支持通过文件选择器导入图片

3. **尺寸设置**
   - **使用标注线**：在图片上画一条线段，输入实际长度，自动计算图片尺寸
   - **直接输入尺寸**：直接输入图片的宽度和高度
   - 支持保持宽高比

4. **图片操作**
   - 支持旋转图片（每次90度）
   - 支持复制图片
   - 支持重新标注尺寸
   - 支持删除图片

5. **一键排版**：自动排列画布上的图片

6. **导出功能**
   - 导出为PNG或JPG图片
   - 导出为JSON文件
   - 临时保存到本地存储

7. **缩放控制**：支持画布和图片的缩放操作

## 使用方法

### 基本操作流程

1. **设置画布**
   - 在左侧面板选择预设尺寸或设置自定义尺寸
   - 调整PPI、颜色模式和背景颜色

2. **导入图片**
   - 拖拽图片到"拖拽图片到这里"区域
   - 或点击"选择文件"按钮选择图片

3. **设置图片尺寸**
   - **使用标注线**：
     - 在弹出的窗口中选择"使用标注线"
     - 在图片上点击并拖动画一条线段
     - 输入线段的实际长度（毫米）
     - 点击"确认添加"
   - **直接输入尺寸**：
     - 在弹出的窗口中选择"直接输入尺寸"
     - 输入图片的宽度和高度（毫米）
     - 可选择是否保持宽高比
     - 点击"确认添加"

4. **编辑图片**
   - 点击图片选中它
   - 使用顶部工具栏进行旋转、复制、重新标注或删除操作

5. **一键排版**
   - 点击顶部的"一键排版"按钮，自动排列所有图片

6. **导出**
   - 点击顶部的"导出图片"按钮，选择格式和质量
   - 点击"导出JSON"按钮，导出为JSON文件
   - 点击"临时保存"按钮，保存到本地存储

### 快捷键和技巧

- **缩放控制**：
  - 使用鼠标滚轮缩放图片
  - 点击缩放按钮进行放大、缩小和复位
- **图片选择**：点击图片即可选中
- **拖拽移动**：选中图片后拖拽可移动位置

## 本地运行

1. 克隆或下载项目到本地
2. 打开项目目录
3. 启动本地服务器（如 `python -m http.server 8000`）
4. 在浏览器中访问 `http://localhost:8000`

## 注意事项

- 建议使用现代浏览器（Chrome, Firefox, Edge）访问
- 图片尺寸设置时，建议使用标注线方式获得更准确的尺寸
- 导出图片时，较高的PPI会产生更大的文件大小
- 临时保存的数据存储在浏览器本地存储中，清除浏览器数据会丢失这些数据

## 常见问题

### Q: 图片导入后没有弹出尺寸设置窗口？
A: 请检查浏览器是否阻止了弹出窗口，或尝试刷新页面后重新导入。

### Q: 标注线设置尺寸时，线段长度不准确？
A: 请确保线段绘制时尽量保持直线，避免弯曲或倾斜。

### Q: 一键排版后图片位置不理想？
A: 一键排版是基于简单的网格排列算法，可能需要手动调整个别图片的位置。

### Q: 导出的图片质量不高？
A: 请在导出时选择较高的JPG质量，或选择PNG格式。
